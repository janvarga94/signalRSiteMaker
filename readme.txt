requirements: npm, visual studio 2017 preview, bower
extensions: npm task runner, gulp task runner

setup automatic bundling on file changes : 
	right click on bundle config and click 'convert to gulp', something like that
	go to npm task runner and install dependencies
	go to gulp task runner, expand it, set 'watch' task to run on project open
